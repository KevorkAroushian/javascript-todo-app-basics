# JavaScript To-Do List App (Beginner Project)

It allows users to add and delete tasks dynamically.


## Project Goal

The goal of this project is to learn:
- How to take user input
- How to create elements dynamically in JavaScript
- How to use event listeners
- How to manipulate the DOM
- How to build a simple interactive web application


## What I learned

- How to create elements using JavaScript
- How to append elements to the DOM
- How to handle button click events
- How to delete elements dynamically
- How to build a simple task management system


## How it works

- User types a task in the input field
- Clicks "Add Task"
- Task is added to the list
- Each task has a "Delete" button
- Clicking delete removes the task
